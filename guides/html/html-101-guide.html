HTML 101 - Learn the Basics

Welcome to the HTML 101 guide. Whether you're starting from scratch or refreshing your knowledge, this guide will walk you through the basics of HTML.

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design webpages. It provides the structure for content like text, images, and videos, allowing web browsers to interpret and display them. HTML uses a system of tags to define elements such as headings, paragraphs, links, and images.

Basic Structure of an HTML Document

Here's an example of the basic structure of an HTML document: 

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Your Page Title</title>
    </head>
    <body>
        <h1>Welcome to HTML!</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Let's break it down:

  • <!DOCTYPE html> - Declares the document type and version of HTML.
  • <html> - The root element that wraps all HTML content.
  • <head> - Contains meta-information like title and character set.
  • <body> - Contains all the content that is visible to users, such as text and images.

Common HTML Tags

HTML uses a wide range of tags to define different types of content. Here are some of the most common ones:

  • <h1>, <h2>, <h3> - Heading tags (from largest to smallest).
  • <p> - Defines a paragraph of text.
  • <a> - Defines a hyperlink to another page.
  • <img> - Defines an image.
  • <ul> <li> - Defines an unordered list and list items.
